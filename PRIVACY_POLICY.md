# プライバシーポリシー

VStoVSC（以下「本アプリ」といいます）の制作者は、本アプリを利用するユーザーのプライバシーを尊重し、個人情報の保護に努めます。

### 1. 個人情報の収集と利用
本アプリ本体は、ユーザーを識別する目的で個人情報を収集・保存・送信しません。
ただし、自動更新時には配信インフラ上で通信に伴う技術情報（IP アドレス、User-Agent 等）が記録される場合があります（詳細は第 3 章をご参照ください）。

### 2. データの取り扱い
本アプリは、ユーザーが指定した Visual Studio ソリューションファイル（.sln / .slnx）を解析し、VS Code 用の設定ファイル（.vscode フォルダ内）を生成します。
*   解析およびファイル生成の処理はすべてユーザーのローカルデバイス上で行われます。
*   読み込んだファイルの内容や、生成された設定データが外部に送信されることはありません。

### 3. 自動更新
本アプリは、起動時に最新バージョンの有無を確認するため、配信元（`https://vs2vsc.nephilim.jp` / Cloudflare R2）から更新マニフェスト（`releases.win.json`）および更新パッケージ（`*.nupkg`）を HTTPS で取得します。
*   配信元のアクセスログにはリクエスト元 IP アドレスや User-Agent などのインフラ標準ログが残ることがありますが、これらは Cloudflare のプライバシーポリシーに基づいて取り扱われ、制作者が個人を特定できる情報を取得することはありません。
*   ユーザー識別子・アカウント情報・利用統計などは一切送信されません。

### 4. お問い合わせ
本プライバシーポリシーまたは本アプリに関するお問い合わせは、GitHub リポジトリの Issue 等を通じて受け付けております。

### 5. 改定
本プライバシーポリシーは、アプリの機能追加や法令の変更に伴い、予告なく改定されることがあります。
