# VStoVSC

Visual Studio のソリューション（.sln / .slnx）を VS Code 用に変換するための Windows 向けデスクトップアプリです。
指定したソリューションの近くに `.vscode` フォルダを生成し、VS Code から MSBuild タスクの実行やデバッグができるようにします。

## 主な機能

- **かんたん操作**: ソリューションファイルをアプリにドラッグ＆ドロップするか、画面をクリックしてファイルを選択するだけで変換を開始できます。
- **最新形式への対応**: 従来の `.sln` 形式だけでなく、新しい `.slnx` 形式のソリューションファイルにも対応しています。
- **タスクの自動生成**: `.vscode/tasks.json` を生成し、以下の操作を VS Code から実行可能にします。
  - ビルド（Debug / Release）
  - クリーン
  - リビルド
- **デバッグ設定の自動生成**: 実行可能プロジェクト（Exe）を検出し、`.vscode/launch.json` を生成します。これにより、VS Code 上で F5 キーによるデバッグが可能になります。
- **自動更新**: アプリ起動時に最新バージョンを確認し、新しいバージョンが見つかった場合はバックグラウンドでダウンロードし、次回以降の起動時に適用されます（通信状態などにより遅れる場合があります）。

## 動作環境

- **OS**: Windows
- **ランタイム**: .NET 10.0

## インストール

[https://vs2vsc.nephilim.jp/VStoVSC-win-Setup.exe](https://vs2vsc.nephilim.jp/VStoVSC-win-Setup.exe) からインストーラ (Setup.exe) をダウンロードして実行してください。スタートメニューとデスクトップにショートカットが作成されます。

インストール後は通常、自動更新で最新版へ追従できます。更新が適用されない場合のみ、インストーラの再取得を検討してください。

## 使い方

1. アプリを起動します。
2. 変換したいソリューションファイル（`.sln` または `.slnx`）をアプリの画面中央にドラッグ＆ドロップします。
   - 画面をクリックしてファイルを選択することも可能です。
3. 変換が完了すると、ソリューションファイルと同じフォルダに `.vscode` フォルダが作成されます。
4. VS Code でソリューションのフォルダを開くと、ビルドやデバッグの機能が利用できるようになります。

## 注意事項

- 既存の `.vscode` フォルダがある場合、削除して再作成するか、既存の設定を保持して `tasks.json` のみを更新するかを選択できます。
- デバッグ設定（`launch.json`）は、ソリューション内に実行可能プロジェクト（OutputType が Exe または WinExe）が特定できた場合にのみ生成されます。
- Visual Studio のインストール状況によっては、初回起動時や変換時に MSBuild の検出に時間がかかる場合があります。
